# Smartphone Market Study Dashboard — Colombia & Global (Telephones / Mobile Devices)

**Live Demo (GitHub Pages):** https://myriskexp.github.io/estudiomercado_TelefonosCelulares/

An interactive strategic market study (dashboard + narrative) covering the **global smartphone industry** and the **Colombian market**, presented as an executive-friendly web artifact.

This repository was developed by **MyRiskExp** to translate market research into a clear, decision-ready site.

---

## What’s inside

### Dashboard / Report modules
- **Executive Summary**
- **TAM / SAM / SOM + CAGR**
- **Key Market Trends** (on-device AI, premiumization, 5G, foldables, fragmentation, e-commerce)
- **Segmentation** (price tiers, OS, regions, Colombia brand dynamics)
- **Competitor landscape**
- **Porter’s Five Forces**
- **Entry barriers, risks & opportunities**
- **Strategic recommendations**

---

## Tech stack
- **React (JSX)** + **Vite**
- Optional (depending on the project version): **Recharts** and **Tailwind CSS**

Dashboard interactivo de inteligencia estratégica que presenta un **estudio de mercado** de la industria de smartphones, cubriendo el **mercado global** y el **mercado colombiano**, en un formato web listo para consumo ejecutivo.

Este repositorio fue desarrollado por **MyRiskExp** para convertir investigación de mercado en un sitio claro y orientado a la toma de decisiones.

---

## Contenido

### Módulos del dashboard / reporte
- **Resumen ejecutivo**
- **TAM / SAM / SOM + CAGR**
- **Tendencias clave** (IA on-device, premiumización, 5G, plegables, fragmentación, e-commerce)
- **Segmentación** (rangos de precio, OS, regiones, dinámica de marcas en Colombia)
- **Panorama competitivo**
- **5 Fuerzas de Porter**
- **Barreras de entrada, riesgos y oportunidades**
- **Recomendaciones estratégicas**
