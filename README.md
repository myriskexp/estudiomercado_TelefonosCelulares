An interactive strategic market study (dashboard + narrative) covering the **global smartphone industry** and the **Colombian market**, presented as an executive-friendly web artifact.

This repository was developed by **MyRiskExp** to translate market research into a clear, decision-ready site.

---

## What’s inside

### Dashboard / Report modules
- **Executive Summary**
- **TAM / SAM / SOM + CAGR**
- **Key Market Trends** (on-device AI, premiumization, 5G, foldables, fragmentation, e-commerce)
- **Segmentation** (price tiers, OS, regions, Colombia brand dynamics)
- **Competitor landscape**
- **Porter’s Five Forces**
- **Entry barriers, risks & opportunities**
- **Strategic recommendations**
